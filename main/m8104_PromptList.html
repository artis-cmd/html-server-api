## 요청 사항
회의록 議事録 의 내용을 제약 조건에 따라 정리해주세요

## 제약 조건
- 회의록의 내용에 따라 정리할 것
- 발화 내용에서 발화 의도를 포함한 하고 싶은 말을 추출할 것
- 추출한 내용에서 연관된 것들을 종합해 하나의 요점으로 정리할 것
- 각각의 요점은 100자 정도로 요약할 것
- 명확하고 의미를 이해하기 쉬운 문장으로 만들 것
- 요점을 Bulletpoint로 나타낼 것

4. To do
