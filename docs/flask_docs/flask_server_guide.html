1. 서버 실행 & 관리 2. 문제 해결 가이드 3. 검색 엔진 관리 4. 자동 실행 설정

1. 서버 실행 & 관리

Flask 서버를 실행하는 방법:

python app.py

2. 문제 해결 가이드

서버가 작동하지 않을 경우 아래를 확인하세요:

  • 포트 충돌 여부 확인 (netstat -tulnp)
  • 의존성 문제 해결 (pip install -r requirements.txt)

3. HTML 문서 검색 엔진 관리

검색 엔진이 정상적으로 작동하지 않을 경우:

tail -f search_engine.log

4. 자동 실행 설정

서버 자동 실행 설정:

systemctl enable flask.service
