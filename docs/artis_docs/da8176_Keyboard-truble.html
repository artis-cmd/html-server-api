🛠 Windows 오른쪽 Alt 키 한/영 전환 문제 해결 가이드

문제 요약

  • 오른쪽 Alt(RAlt) 키로 한/영 전환이 안 됨
  • RAlt 키를 눌렀을 때 Chrome 등에서 엉뚱한 기능(예: 점 세 개 메뉴) 실행
  • Alt + A로 á 같은 특수문자도 입력되지 않음

원인 가능성

  1. IME 입력기 문제 (Microsoft IME 초기화 필요)
  2. 키보드 드라이버 오류 또는 충돌
  3. Scancode Map 레지스트리 값으로 인해 키가 무력화됨
  4. AutoHotKey, PowerToys, 키보드 제어 유틸 등 외부 툴의 간섭
  5. 사용자 계정 또는 Windows 시스템 환경 자체의 충돌

단계별 해결 방법

① 입력기 확인

  • Microsoft IME만 남기고 나머지 키보드 레이아웃 제거
  • IME 우클릭 → Microsoft IME로 설정되어 있는지 확인

② Scancode Map 레지스트리 점검

경로: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout
→ "Scancode Map" 항목이 있다면 삭제하고 재부팅

③ 키보드 드라이버 초기화

  • 장치 관리자 > 키보드 > HID 키보드 디바이스 전부 제거
  • PC 재부팅 → 자동 재설치

④ AutoHotKey 우회 설정 (권장)

  • AutoHotKey v1.1 설치 후 아래 스크립트 저장 및 실행
#Requires AutoHotkey v1.1
RAlt::
Send, {vk15sc138}
Return
  • 트레이에 초록색 H 아이콘 표시 → 정상 실행 중
  • RAlt 누르면 Hangul 키 직접 전송 → 한영 전환 가능

⑤ Windows 시작 시 자동 실행 등록 (선택)

  • Win + R → shell:startup
  • 해당 폴더에 `.ahk` 파일의 바로가기(.lnk) 추가

⑥ (선택) 한국어 언어팩 제거 및 재설치

  1. 설정 > 시간 및 언어 > 언어 및 지역
  2. 시스템 언어를 영어 또는 일본어로 변경
  3. 한국어 언어팩 삭제 후 다시 추가

최종 확인

  • 메모장에서 RAlt 키로 "가" ↔ "A" 전환됨
  • Chrome 등에서 더보기 메뉴 안 열림
  • 입력기 전환 단축키도 정상 작동 (Alt+Shift / Ctrl+Shift)
