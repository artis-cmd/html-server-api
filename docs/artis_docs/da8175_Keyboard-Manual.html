📱 블루투스 연결 관련
기능 단축키 설명
블루투스 등록 Fn ✚ Q ➡️ Fn + Ctrl + #1~4 페어링 모드 진입 후 등록할 슬롯 선택
블루투스 삭제 Fn ✚ Q ➡️ Fn ✚ Ctrl ✚ Del ✚ #️⃣1 ~ 4 키보드 롬에 등록된 특정 블루투스 기기 삭제
모든 블루투스 삭제 Fn ✚ Q ➡️ Fn ✚ Z ✚ Del 등록된 모든 블루투스 기기 삭제
블루투스 전환 Fn ✚ Ctrl ✚ #️⃣1 ~ 4 등록된 1 ~ 4번 블루투스 기기로 전환
USB 유선연결 전환 Fn ✚ Ctrl ✚ #️⃣0 유선 연결 모드로 전환 (소프트웨어 사용 시 필요)
페어링 모드 취소 Fn ✚ X 페어링 모드(Fn + Q) 취소

사용 안내

※ ➡️ 표시는 다음 단계를 의미합니다.

※ #️⃣ 표시는 0, 1, 2, 3, 4 단일 숫자를 의미합니다.

※ ✚ 표시는 순서대로 같이 눌러야 한다는 의미입니다.

💻 윈도우에서 블루투스 등록 방법
  1. 윈도우 ➡️ 설정 ➡️ 장치 ➡️ Bluetooth 또는 기타 장치 추가 ➡️ Bluetooth(마우스, 키보드, 팬....) 클릭
  2. 해피해킹 키보드 전원을 켜고 Fn ✚ Q로 페어링 모드로 진입
  3. Fn ✚ Ctrl ✚ #️⃣1 ~ 4 (등록하고 싶은 숫자 선택)
  4. 윈도우 화면에 "HHKB_Hybrid_1"과 같은 형식으로 키보드 이름이 표시됨
  5. 표시된 장치를 클릭하고 숫자 입력 후 엔터 키를 누르면 등록 완료
