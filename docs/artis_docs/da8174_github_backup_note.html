📁 GitHub 백업 및 공개/비공개 설정 방법

1. GitHub 저장소에 백업하는 방법

  • git init으로 로컬 저장소 초기화
  • git remote add origin https://github.com/USERNAME/REPO.git 연결
  • git add . 전체 변경 파일 스테이징
  • git commit -m "commit message" 커밋
  • git push origin main GitHub로 업로드

2. 공개/비공개 설정 방법

  • GitHub 저장소 > Settings > General > Visibility 항목
  • Public 선택 시 누구나 접근 가능
  • Private 선택 시 로그인된 사용자만 접근 가능
  • 변경 즉시 저장소 접근 권한이 반영됨

3. .bat 파일로 자동 백업

  • 다음과 같이 git-backup.bat 파일 생성
    • @echo off
cd /d E:\OneDrive\HTML\html-server-api
echo 🔄 변경사항 스테이징 중...
git add .
set /p msg=📝 커밋 메시지를 입력하세요: 
git commit -m "%msg%"
echo 🚀 GitHub로 푸시 중...
git push origin main
echo ✅ 완료! 창을 닫아주세요.
pause
  • 더블클릭만 하면 자동 백업 완료!
