Eagle 주요 단축키 정리

⌨️ 자주 쓰는 기능 위주로 정리했으며, Mac과 Windows 공통 단축키를 기준으로 작성됨.

📸 캡처 관련

  • 선택 영역 캡처: Ctrl + Alt + E
  • 창 캡처: Ctrl + Alt + W
  • 전체 화면 캡처: Ctrl + Alt + A

📁 정리 및 분류

  • 태그 추가: T
  • 컬러 라벨 추가: Shift + 1~9
  • 항목 삭제: Delete 또는 Backspace

🔍 탐색 및 보기

  • 검색창 열기: Ctrl + F
  • 전체 보기: Ctrl + 1
  • 태그 보기: Ctrl + 2
  • 컬러 보기: Ctrl + 3

🖼 이미지 작업

  • 원본 이미지 열기: Space
  • 슬라이드 쇼 시작: Ctrl + Enter
  • 이전 / 다음 항목: ← / →

📌 참고: 단축키는 Eagle 설정 > 단축키 탭에서 직접 변경 가능하며, 일부 OS에서는 권한 설정이 필요할 수 있음.
